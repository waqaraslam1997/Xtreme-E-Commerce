
# Xtreme Commerce App

Lets dive to the Xtreme World!

## Features

- Light/dark mode toggle
- Sorting (high to low / low to high) price
- Filtering Products by Category, Size & Color
- Searching Products
- Cross platform
- Stripe Payment
- Sanity (Backdoor for adding Products easily with UI)
